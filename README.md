# Blogicum — платформа для ведения блога

Django-проект блог-платформы с возможностью публикации постов, комментирования и управления профилем пользователя.

## Функционал

- **Публикации**: создание, редактирование, удаление постов с изображениями
- **Категории**: тематическая категоризация публикаций
- **Комментарии**: возможность комментировать посты (только для авторизованных)
- **Профили пользователей**: страница профиля с историей публикаций
- **Регистрация и аутентификация**: встроенная система пользователей Django
- **Пагинация**: по 10 публикаций на страницу
- **Отложенные публикации**: возможность планировать посты на будущее

## Технологии

- Django 3.2.16
- Python 3.10+
- SQLite (для разработки)
- django-bootstrap5

## Быстрый старт

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Примените миграции:
   ```bash
   cd blogicum
   python manage.py migrate
   ```

3. Загрузите тестовые данные (опционально):
   ```bash
   python manage.py loaddata ../db.json
   ```

4. Создайте суперпользователя:
   ```bash
   python manage.py createsuperuser
   ```

5. Запустите сервер разработки:
   ```bash
   python manage.py runserver
   ```

Проект доступен по адресу `http://127.0.0.1:8000/`

## Структура проекта

```
blogicum/
├── blog/           # Приложение с публикациями, комментариями
├── pages/          # Статические страницы (О проекте, Правила)
├── templates/      # HTML-шаблоны
├── static/         # Статические файлы (CSS, JS, изображения)
└── media/          # Загруженные пользовательские файлы
```

## Админ-панель

Доступна по адресу `/admin/`. Для входа используйте учётные данные суперпользователя.

В админке можно управлять:
- Публикациями
- Категориями
- Местоположениями
- Комментариями
